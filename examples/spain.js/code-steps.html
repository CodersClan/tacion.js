Using steps to define sequence

  • Steps allow for multi-part slides, and provide a framework for sequencing.
  • To define a step, add the data attribute to any DOM node in the content markup:
  • <div data-step="1"></div>
  • As each step is reached, tacion adds a class of active to the node.
  • This allows the implementor to define custom presentational logic for each step. However, the implementer can also use tacion's built-in presentational logic:
  • <div class="step opacity" data-step="1"></div>
