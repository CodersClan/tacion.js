Running the server

  • The server is a nodejs application that talks to Pusher's secure REST API.
  • To facilitate this communication, the server rely's on Jaewoong Kim's REST API client:
$> npm install node-pusher
  • Once installed, running the server is as easy as starting any other node process:
$> node src/server/tacion.js [app_id] [api_key] [api_secret]
  • You can find any of the required server parameters on your corresponding app page
